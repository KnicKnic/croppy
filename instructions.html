Purpose

  • Scales and splits long form comics into Webtoon and Tapas format

Features

  • Website works offline. Once visited you do not need internet for this website.
  • Uses webassembly to run ImageMagick in your browser.
  • You can hover over split images to find out where one image ends.

Troubleshooting

  • Very large files 50MB+ will probably not work, I suggest you use ImageMagick not in your browser
  • Use a browser with high webassembly compatibility like Chrome or Firefox.
  • Report issues at https://forums.tapas.io/t/free-image-slicer/24139, mention version 1_3

Manual links

Since croppy uses imagemagick, here are some links to simliar imagemagick commands that croppy uses. Croppy does a few extra checks and will scale down your picture if its too big. However these links may be useful if you wish to tweak croppy settings

  • https://bit.ly/croppy-resize Size a picture to 800 width. Since this is only a resize it may succeed if a full croppy action fails. You will have to feed the output of this link into croppy.
  • https://bit.ly/wasm-imagemagick-webtoon Resize for webtoons using jpeg
  • https://bit.ly/wasm-imagemagick-tapas Resize for tapas using png
  • https://bit.ly/wasm-imagemagick-webcomicsapp Resize for webcomicsapp using jpeg
