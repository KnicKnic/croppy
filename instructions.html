Features

  • Webtoons Extension - Works on Desktop & Mobile even iPad
    • No Zips!
    • Extension instructions be sure to checkout the videos.
  • Website works offline. Once visited you do not need internet for this website.
  • Uses webassembly to run ImageMagick in your browser.
  • You can hover over split images to find out where one image ends.

Troubleshooting

  • Very large files 50MB+ will probably not work, I suggest you use ImageMagick not in your browser
  • Use a browser with high webassembly compatibility like Chrome or Firefox.
  • Report issues at https://forums.tapas.io/t/free-image-slicer/24139, mention version 1_3
